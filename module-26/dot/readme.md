 Accessing object data: nested object, dot vs bracket and optional chaining

1️⃣ Nested Object
Nested Object মানে হলো
👉 একটা object এর ভিতরে আরেকটা object থাকা।


2️⃣ Dot Notation (.)
Dot দিয়ে object data access করা হয়।


3️⃣ Bracket Notation ([])
Bracket notation ব্যবহার করা হয় যখন:

✔ key dynamic
✔ key variable
✔ key এ space থাকে


4️⃣ Optional Chaining (?.)
Optional chaining ব্যবহার করা হয়
👉 error না হওয়ার জন্য

ধরো object এর ভিতরে data নেই।


Quick Note:

Object Access:
Dot Notation → object.key
Bracket Notation → object["key"]

Nested Object:
object.key.subkey

Optional Chaining:
object?.key?.subkey

👉 Error এড়াতে ব্যবহার করা হয়।
